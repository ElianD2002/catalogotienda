# Chicharronera Tío Jaime — Landing Page & Menú Digital 🐷🔥

Página de aterrizaje optimizada y menú digital autogestionable para la **Chicharronera Tío Jaime** ubicada en Olanchito, Yoro, Honduras. Diseñada para un rendimiento web móvil excepcional y facilidad de administración en tiempo real.

---

## 🛠️ Stack Tecnológico

El proyecto está construido con un stack de desarrollo moderno y liviano:

*   🚀 **Astro (v6):** Framework de frontend enfocado en velocidad y generación estática estricta.
*   📝 **Tina CMS (v3):** Panel de administración visual Git-backed para gestionar el contenido local o en la nube.
*   🎨 **Tailwind CSS (v4):** Estilos fluidos, responsivos y con animaciones de micro-interacción integradas en `@tailwindcss/vite`.
*   💬 **WhatsApp Integration:** Generación dinámica de mensajes preestablecidos para ordenar comida con un solo click.

## 🚀 Comandos de Desarrollo

Todas las operaciones se corren desde la raíz del proyecto usando Node/NPM:

| Comando | Acción |
| :--- | :--- |
| `npm install` | Instala todas las dependencias del proyecto. |
| `npm run dev` | Inicia el entorno local de Astro y Tina CMS. |
| `npm run build` | Compila el sitio estático optimizado en la carpeta `./dist/`. |
| `npm run preview` | Previsualiza localmente el compilado final de producción. |

### Acceso al Panel de Control
Cuando el servidor de desarrollo está activo (`npm run dev`), puedes acceder al panel visual de administración en la siguiente ruta local:
🔗 [http://localhost:4321/admin](http://localhost:4321/admin)
